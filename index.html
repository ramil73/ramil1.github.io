Language and Tools Knows

Python

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation

Java

Java is a high-level, object-oriented programming language designed to be platform-independent, allowing developers to write code once and run it anywhere using the Java Virtual Machine (JVM).

Figma

Figma is a cloud-based design and prototyping tool that allows users to create, collaborate, and share user interface designs in real time.

Html

HTML is the standard language used to create and structure content on the web. It defines elements like headings, paragraphs, images, links, and more, allowing browsers to display web pages correctly

Css

CSS is a language used to control the appearance and layout of HTML elements on a web page. It defines styles like colors, fonts, spacing, and positioning, allowing developers to create visually appealing and responsive designs.
